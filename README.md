# 🎯 LeadPulse: Strategic Sales Intelligence

## 🚀 Project Overview
LeadPulse is a full-stack Machine Learning application designed to prioritize sales leads using predictive analytics. By analyzing historical interaction data, the system identifies "Hot Leads" with a high probability of conversion, allowing sales teams to optimize their workflow.

## 🛠️ Tech Stack
- **Machine Learning:** Random Forest Classifier (Scikit-Learn)
- **MLOps:** MLflow for experiment tracking
- **Frontend:** Streamlit & Plotly
- **Environment:** Python 3.11

## 📈 Key Features
- Real-time lead scoring dashboard.
- Behavioral influence analysis (identifying what drives a score).
- Strategic directives based on model confidence.
